# 行動規範 (Code of Conduct)

## 我々の約束

SATORI プロジェクトのメンバーとして、より開かれた、参加しやすいコミュニティを築くため、
私たちは、年齢、体格、障害、民族性、性別、性自認・男女差別のない、
経験レベル、国籍、個人的な外見、人種、宗教、性的同一性と性的指向に関わらず、
全ての人にハラスメントなく参加できる環境を提供することを約束します。

## 我々の基準

SATORI コミュニティに属する者として、以下のことを約束します:

- **敬意を持つ**:
  異なる意見、経験、背景を尊重します。建設的な批判を受け入れます。
  他者を尊重する姿勢で議論します。

- **包括的である**:
  すべての貢献者を歓迎します。初心者から専門家まで、多様な視点を価値あるものとして扱います。

- **透明性を保つ**:
  コミュニケーションは明確で、正直で、直接的であるべきです。

- **協力的である**:
  共通の目標—科学データ分析の民主化—に向けて協力します。

- **建設的なフィードバック**:
  他者のコードやアイデアに対する批判は、常に建設的で改善を目指したものであるべきです。

## 許容されない行為

以下の行為は許容されません:

- ハラスメント、ヘイトスピーチ、または差別的な言動
- アイデアの窃盗や過度な帰属請求
- 個人情報の無断公開（ドキシング）
- 物理的な暴力、その脅迫、またはそうした行為への扇動
- スパム、トローリング、不誠実な行動
- 他者に対する一貫したハラスメント

## 報告と改善

### 問題を報告する

このコード・オブ・コンダクトに違反する行為を目撃した場合は、
プロジェクト管理者 ([@nahisaho](https://github.com/nahisaho)) に
以下の方法で報告してください:

1. **GitHub 上で非公開に**: Issue を作成せず、
   [GitHub Security Advisory](https://github.com/nahisaho/satori/security/advisories) から報告してください。

2. **メール**: nahisaho.satori [at] gmail.com に直接ご連絡ください。

3. **一対一での相談**: GitHub でダイレクトメッセージを送ってください。

報告する際は、以下の情報を含めてください:

- 問題の詳細な説明
- それが発生した時期と場所（Issue、PR、Discussion、チャットなど）
- 関係者の GitHub アカウント名（可能な場合）
- あなたが知っている関連する詳細
- あなたの連絡先情報

**すべての報告は機密扱いされます。** 
報告者の同意がない限り、その身元は公開されません。

### 対应と処分

管理者は報告されたすべての申し立てを調査します。

状況に応じて、以下の対応が取られる可能性があります:

1. **警告**: 軽微な初回の違反に対する注意と改善の要求
2. **一時的な制限**: コミュニティ参加の制限（指定期間）
3. **永久的な除外**: プロジェクトから削除（深刻または繰り返される違反の場合）

### 異議申立て

決定に異議がある場合は、管理者に書面で異議を唱えることができます。
異議申立てを含む申し立ては、機密に調査されます。

## 対象範囲

この行動規範は、以下の場所に適用されます:

- GitHub リポジトリ（Issues、PR、Discussion、Wiki）
- Discord・Slack などのコミュニティチャット
- メールでのコミュニケーション
- SATORI 関連のイベント・会議
